## 🚀 Quick Start

### Access the Texture Gallery
1. Open your 3D Parametric Engine
2. Press **X** key or click the **"Textures"** button
3. Browse the gallery and select quality tier
4. Download your favorite textures

### Apply to 3D Models
1. Create a parametric curve (torus knot, hypocycloid, etc.)
2. Export as **GLB** format (includes full metadata)
3. Open in Blender/Maya/3ds Max
4. Apply downloaded textures as diffuse/albedo maps
5. Set texture filtering to **"Nearest"** for crisp pixels
6. Re-export your textured model

## 📖 Usage Examples

### Blender Workflow
```python
# Quick setup in Blender
1. Import your parametric GLB model
2. Add Image Texture node to shader
3. Load downloaded texture
4. Connect to Base Color
5. Set Interpolation to "Closest" (for pixels)
6. Adjust Emission for glow (optional)
7. Render and enjoy!
```

### Unity/Unreal Setup
```
Unity:
- Import texture → Project folder
- Set Filter Mode to "Point (no filter)"
- Apply to material as Albedo
- Adjust smoothness/metallic

Unreal Engine:
- Import texture → Content Browser
- Set Compression to "UserInterface2D"
- In material, set Sampler Source to "Shared: Wrap"
- Apply to Base Color
```
