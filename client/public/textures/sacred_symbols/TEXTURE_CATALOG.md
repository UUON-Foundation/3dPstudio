## Overview

This collection features 18 handcrafted pixel art textures across 6 sacred symbols, each available in 3 quality tiers. Designed specifically for 3D model texturing, game assets, and spiritual visualization projects.

## Texture Quality Tiers

### Tier 1: 8-bit Retro Style
- **Color Palette:** 16 colors maximum
- **Shading:** Flat shading with dithering
- **Style:** Classic NES/early game console aesthetic
- **Use Case:** Retro games, low-poly art, nostalgia projects

### Tier 2: 12-bit Enhanced Style
- **Color Palette:** 32 colors
- **Shading:** Basic light wrapping and soft shadows
- **Style:** Mid-era retro gaming (Genesis/SNES early)
- **Use Case:** Balanced detail for modern retro aesthetics

### Tier 3: 16-bit Premium Style
- **Color Palette:** 64 colors with smooth blending
- **Shading:** Rim lighting, soft shadows, gradient effects
- **Style:** SNES/Sega Saturn era high-quality pixel art
- **Use Case:** Professional game assets, promotional materials

## Technical Specifications

### Resolution
- All textures: 1024x1024 pixels
- Image rendering: Pixelated (crisp pixel edges preserved)

### Format
- File type: PNG with transparency support
- Color space: sRGB
- Bit depth: 8-bit per channel

### Naming Convention
```
[symbol_name]_diffuse_t[tier_number].png
```

Example: `chakra_mandala_diffuse_t3.png`

## Usage Guidelines

### For 3D Applications
1. **Import** textures into your 3D software (Blender, Maya, 3ds Max)
2. **Apply** as diffuse/albedo maps to your geometry
3. **Set** texture filtering to "Nearest" to preserve pixel aesthetic
4. **Export** your models with textures embedded (GLB format recommended)

### For the Parametric Engine
1. Press **X** or click **Textures** button to open the gallery
2. Select your desired quality tier
3. Download textures for external use
4. Apply to parametric curves exported as 3D models

### Recommended Uses
- 3D model texturing for games
- VR/AR spiritual experiences
- NFT art and digital collectibles
- Educational materials on sacred geometry
- Meditation and wellness apps
- Promotional materials for spiritual brands

## PBR Material Setup (Advanced)

For realistic rendering, combine these diffuse textures with:
- **Normal Map:** Create from diffuse using photo editing software
- **Roughness Map:** 0.4-0.6 for slight shine
- **Metallic Map:** 0.0-0.3 for subtle metallic highlights
- **AO (Ambient Occlusion):** Enhance depth in recessed areas
- **Emission Map:** Add glow to sacred symbols (optional)
